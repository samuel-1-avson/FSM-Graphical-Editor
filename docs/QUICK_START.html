Quick Start Guide - BSM Designer

Welcome to the Brain State Machine Designer!

Basic Operations

  • Adding States & Comments: Use the "Add" tools from the toolbar or drag items from the "Elements Palette" dock.
  • Adding Transitions: Use the "Add Transition" tool and click on a source state, then a target state.
  • Editing Properties: Double-click any item, or select it and use the "Item Properties" dock for quick edits. Click "Advanced Edit..." for more options.
  • Saving/Loading: Use the File menu (File > Save, File > Open).

New: Hierarchical State Machines & Breakpoints

You can now create complex FSMs by turning a state into a Superstate (in its properties). Double-click the superstate or use the "Edit Sub-Machine..." button to design its internal FSM. During Python simulation, you can now also right-click a state > Toggle Breakpoint to pause execution when that state is entered.

Simulation & Code Generation

  • Python Simulation: Use the "Python Simulation" dock to start, stop, reset, and step through your FSM. Trigger events and observe variable changes in real-time.
  • Code Generation: Use the File > Export menu to generate Simulink models, C code stubs, or a complete Python FSM class.

New: Integrated Development Environment (IDE)

Access a full code editor via View > Docks & Panels > Standalone Code IDE. Write, save, and run Python scripts directly within the application, perfect for testing concepts or managing helper scripts.

AI Assistant (Google Gemini)

Configure your Google AI API Key in "AI Assistant" > "AI Assistant Settings...". The assistant uses Google's Gemini model to help you generate FSMs from descriptions, explain code, or help debug.

New: Full Theming and Customization

Go to Edit > Preferences... to completely customize your experience.

  • Appearance: Switch between Light and Dark themes.
  • Item Defaults: Set the default look (shape, font, border, color) for all new states, transitions, and comments.
  • Behavior: Enable or disable the new live Resource Monitor in the status bar.

Organize your workspace using View > Perspectives to save and load different dock layouts for design, simulation, or coding.
